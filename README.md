# Shopify Theme (Figma-targeted storefront)

This repository contains a complete Shopify Online Store 2.0 theme scaffold with reusable, customizable sections designed for a modern ecommerce landing page:

- Announcement bar + sticky header
- Hero with dual CTAs
- Featured collection grid
- Split image/text feature section
- Value proposition cards
- Testimonials
- Rich text CTA
- Footer

## Preview locally

```bash
shopify theme dev
```

Then open Shopify Theme Editor to map fonts, spacing, and assets exactly to the target Figma.
